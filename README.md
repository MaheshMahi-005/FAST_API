--> Install the virtual environment on IDE or Where you are running this file using : " python -m venv name " # name -- add your chosed names.<br/>

--> Then run the virtual enivornment by using it's path present on the venv folder or use " .\name\Scripts\python.exe ".<br/>

--> And then install the requirements.txt by : " pip install -r requirements.txt ".<br/>

--> Run the server : " uvicorn main:app --reload ".<br/>

--> open the url and you see the data in a Json format.<br/>

--> We can Test this by using Swagger UI, at the end of the URL add '\docs' to open the Swagger UI .<br/>
